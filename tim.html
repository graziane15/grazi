COMO ELES AGEM

Como funciona o hacking? Como agem os hackers? Os hackers usam uma variedade de técnicas para alcançar seus objetivos criminosos. Alguns métodos comuns incluem: Engenharia social Engenharia social é uma técnica de manipulação projetada para explorar o erro humano para ter
acesso a informações pessoais. Usando uma identidade falsa e vários truques emocionais,
os hackers enganam as pessoas e as fazem divulgar informações pessoais ou financeiras. Eles podem usar golpes de phishing,
e-mails de spam ou mensagens instantâneas ou até mesmo em sites falsos para isso. Hacking de senhas Os hackers usam modos diferentes para obter senhas. O método de tentativa
e erro é chamado de ataque de força bruta, onde os hackers tentam adivinhar todas as combinações possíveis para obter acesso. Os hackers também podem usar algoritmos simples para gerar diferentes combinações de letras, números e símbolos para facilitar a identificação de combinações de senhas. Outra técnica é o ataque de dicionário, um programa que insere palavras comuns em campos de senha para acertar eventualmente uma senha. Infectando dispositivos com malware Hackers podem se infiltrar no dispositivo de um usuário e instalar malwares.
É provável que tenham como alvo vítimas potenciais via e-mail, mensagens instantâneas e sites com conteúdo para download ou redes de transferência
de dados ponto a ponto. Explorando redes sem fio desprotegidas Em vez de usar códigos maliciosos para se infiltrar no computador de alguém, os hackers podem simplesmente se
aproveitar de redes sem fio abertas. Nem todos protegem seu roteador. Os hackers podem explorar isso, procurando uma conexão sem fio aberta e sem segurança.
Isso é conhecido como wardriving Após se conectarem à rede desprotegida, os hackers só precisam contornar a segurança básica para obter acesso a dispositivos conectados a essa rede. Obtendo acesso via backdoor Os hackers podem criar programas que buscam caminhos desprotegidos em sistemas de rede e
computadores. Eles podem obter acesso via backdoor, infectando um computador ou sistema com um trojan, um tipo de vírus criado por hackers para adquirir e roubar dados importantes de forma discreta. Espionando e-mails Os hackers podem criar código para interceptar e ler e-mails. A maioria dos programas
de e-mail hoje usam fórmulas de criptografia. Com isso, mesmo que os hackers interceptem uma mensagem, não poderão lê-la. Gravação de teclas Alguns programas permitem aos hackers rastrear cada tecla apertada no computador do usuário.
Após serem instalados no computador da vítima, os programas gravam as teclas usadas, dando ao hacker tudo o que precisam para se infiltrar em um sistema ou roubar a identidade de alguém. Criando máquinas zumbis Um computador zumbi, ou bot, é um computador usado pelo hacker para enviar spam ou gerar ataques DDoS
(Negação Distribuída de Serviço). Depois que uma vítima executa um código aparentemente inofensivo, é criada uma conexão entre o seu computador e o sistema do hacker. O hacker passa a controlar secretamente o computador da vítima, usando-o para cometer crimes ou espalhar spam.
